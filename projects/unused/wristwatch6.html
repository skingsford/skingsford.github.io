Wristwatch Animation

To Display Actual Time

This is a self-promotional exercise displaying a number of disciplines to create a unique demonstration of what can be achieved using a diverse digital toolset.

The wristwatch model was designed and rendered in 3D with the hand animation driven by the clock of the device on which it is being viewed.

The original was animated with Adobe Flash which is visually sharper. This version is HTML5 to ensure it works on any modern web-enabled device.
